# 🧹 CleanPro — Сайт-визитка клининговой службы

Лендинг для клининговой службы, созданный на **React + Vite**. Современный минималистичный дизайн с акцентными цветами.

## 🚀 Запуск проекта

### Предварительные требования
- **Node.js** версии 18 или выше ([скачать](https://nodejs.org/))

### Установка и запуск

```bash
# 1. Клонировать репозиторий (или скачать архив)
git clone <url-репозитория>
cd cleaning-service

# 2. Установить зависимости
npm install

# 3. Запустить в режиме разработки
npm run dev
